## Features

- Show all Tickets List
- Show indiviual Ticket
- Create new Ticket
- Modify existing Ticket
- Delete Ticket

## Start the Project

- Start Next.js
  ```
  cd nextjs-task-ticket-app/
  npm run dev
  ```
- Start expo
    ```
    cd react-native-ticket-app/
    npm run start
    ```
